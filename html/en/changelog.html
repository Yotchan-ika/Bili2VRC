Bili2VRC 2.0.0

June 15, 2025

Changes

  • The extension has been reborn as "Bili2VRC"!
  • The UI design and behavior have been completely revamped.

New Features

  • Added a video parsing button directly on the video page. You can now use this extension without pinning it to the extensions bar.
  • Added a video parsing history page for videos.
  • Added a tutorial feature.
  • Video parsing is now handled within the extension itself. This should improve loading speed in VRChat and help prevent errors.
